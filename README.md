# ABC Inventory App
This is a custom AI tool for cataloging shop inventory using Gemini Vision.

### How it works:
- **`index.html`**: The main app code.
- **AI Model**: Gemini 1.5 Flash (via Google AI Studio).
- **Function**: Uploads a photo, identifies the item, and exports the data to a CSV spreadsheet.

### Instructions:
1. Open the live GitHub Pages link.
2. Upload a photo.
3. Click "Auto-Identify."
4. Export the file for your records.
